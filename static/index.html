Algorithms Used

Understanding the six ML algorithms powering our predictions.

1

Random Forest

An ensemble of decision trees that combines multiple weak learners into a strong classifier. Uses bagging (bootstrap aggregating) to reduce overfitting and improve generalization. Excellent for handling non-linear relationships and feature interactions.

2

Support Vector Machine

Finds the optimal hyperplane that maximally separates classes in high-dimensional space. Uses the RBF kernel to handle non-linearly separable data by mapping features to higher dimensions. Robust in high-dimensional spaces with clear margin of separation.

3

K-Nearest Neighbors

A non-parametric, instance-based learning algorithm that classifies based on the majority vote of its K nearest neighbors. Simple yet effective, requiring no explicit training phase. Performance depends on the choice of K and distance metric.

4

Logistic Regression

A linear classifier that models the probability of a binary outcome using the sigmoid function. Despite its simplicity, it provides highly interpretable results with coefficient-based feature importance. Serves as an excellent baseline model.

5

Gradient Boosting

Builds an ensemble of weak decision trees sequentially, where each tree corrects the errors of its predecessor. Uses gradient descent to minimize a loss function. Often achieves the highest accuracy on structured/tabular data among all algorithms.

6

Gaussian Naive Bayes

A probabilistic classifier based on Bayes' theorem with the "naive" assumption of feature independence. Assumes features follow a Gaussian (normal) distribution. Extremely fast and works well with small datasets and high-dimensional data.

About

About HealthGuard AI

Built as an AICTE Capstone Project to demonstrate practical applications of Machine Learning in healthcare.

📋

Project Overview

HealthGuard AI is a multi-disease prediction system that leverages six different machine learning algorithms to predict Heart Disease, Diabetes, and Breast Cancer. It compares algorithm performance and presents results through an interactive dashboard.

📊

Datasets Used

Heart Disease: Based on the UCI Cleveland Heart Disease dataset (920 records, 13 features).
Diabetes: Modeled on the Pima Indians Diabetes dataset (768 records, 8 features).
Breast Cancer: Wisconsin Breast Cancer dataset from scikit-learn (569 records, 30 features).
